# AFS - Archive.org Free Search

A lightweight web tool to search for **publicly available** files on [Archive.org](https://archive.org) — books, documents, and more — using the advanced search syntax.

## Features

- Search for non-restricted (freely accessible) items
- Toggle between opening results in a new tab or redirecting the current page
- Minimal, responsive UI powered by [Pico CSS](https://picocss.com)

## Usage

1. Open [joaoguimaraens.github.io/afs](https://joaoguimaraens.github.io/afs) (or serve `index.html` locally)
2. Enter your search query using Archive.org advanced search syntax
3. Toggle **Redirect** if you prefer results in the current tab
4. Press **Enter** or click the search button
